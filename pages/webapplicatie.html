Opdracht

Bij het vak Advanced Programming Techniques kregen we de opdracht om een eigen webapplicatie met achterliggende database te maken in het Spring Boot framework. Ik besloot om iets te maken wat mijn interesseert: een applicatie waarop een lokale voetbalcompetitie de matchen, klassementen, spelers, ... kan bijhouden. Om niet zelf allemaal data te moeten verzinnen koos ik ervoor om de Jupiler Pro League als voorbeeld te nemen.
